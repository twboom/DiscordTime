What is this?

Almost a year ago, Discord released a feature that allowed users to send a timestamp that would render the local time in the message.

An example use case for this is in announcements, where you want to send a message that contains a time (e.g. "The meeting will be at 12:00"). With the timestamps, everybody sees the time in their local timezone.
